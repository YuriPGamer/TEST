Políticas e Termos

Última atualização: Janeiro de 2026

1. Política de Privacidade

Esta Política descreve como o aplicativo Resultados Finais coleta, usa e protege suas informações, em conformidade com a Lei Geral de Proteção de Dados (LGPD).

1.1. Responsabilidade dos Dados

  • Controlador (Dono dos Dados Escolares): Os dados acadêmicos (notas, faltas, turmas) são de propriedade exclusiva do CEEMTI BAIXO GUANDU.
  • Desenvolvedor (Operador Técnico): O sistema é mantido por Yuri Possimozer, que atua como operador técnico para fornecer a interface de visualização.

1.2. Dados Coletados

  • Identificação: Nome e E-mail (via Login Google) para autenticação.
  • Acadêmicos: Notas, frequência e histórico escolar (somente leitura).
  • Técnicos: Logs de acesso e endereço IP para auditoria de segurança.

1.3. Uso das Informações

Seus dados são utilizados estritamente para:

  • Autenticar seu acesso ao sistema (garantindo que apenas pessoas autorizadas vejam as notas).
  • Exibir o boletim escolar digital.
  • Monitorar a segurança do aplicativo contra acessos indevidos.

Não vendemos, não alugamos e não compartilhamos seus dados com terceiros para fins publicitários.

2. Termos de Uso (Serviço)

2.1. Natureza do Serviço

Este aplicativo é uma ferramenta facilitadora para consulta rápida de resultados escolares. O documento oficial, para todos os fins legais e de transferência, continua sendo o Boletim ou Histórico Escolar impresso e assinado pela secretaria do CEEMTI BAIXO GUANDU.

2.2. Acesso e Segurança

O acesso é restrito. Ao utilizar o sistema, você concorda em:

  • Manter o sigilo das suas credenciais de acesso (Conta Google).
  • Não tentar explorar falhas de segurança no sistema.
  • Utilizar os dados apenas para acompanhamento pedagógico pessoal.

2.3. Isenção de Garantias

O serviço é fornecido "como está". Embora nos esforcemos para garantir a precisão dos dados, o desenvolvedor não se responsabiliza por eventuais divergências causadas por erro de digitação na base de dados original ou interrupções temporárias do serviço (Google Cloud/GitHub).
